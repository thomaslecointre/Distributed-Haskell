Traitement pour l'esclave



L'esclave prend en argument : [String] tel que ["keyword","serie","seasonNumber","numberOfEpisode"]
keyword = "Ned"
serie = "game_of_thrones"
seasonNumber = "1"
numberOfEpisode = "7"

La sortie :
- avec keyword : [[[Int]]] (1er tableau : résultat par esclave, 2e tableau : résultat par saison, 3e tableau : Season, Episode, Occurrence

- sans keyword (N/A = statistique) : à faire ( [[Int, Int, [String, Int]]] par esclave, par saison, saison_episode_nuageDeMots )



Dans traitementv4, case avec le tri souhaité :

Les étapes :
LireArgumentEsclave : crée les url à traiter

/!\ LireURLJson : renvoie le résumé

- GetWordsFilev2 : supprime les stop words
  PorterStemmer : utilise l'algorithme de Porter
  KeyWordOccurrence : pour retourner l'occurence d'un keyword
  Sort : pour trier et créer du json

- Statisticsv2 : pour faire le nuage de mots d'un épisode
